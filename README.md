# Desafios-SQL

## OBJETIVO
O presente repositório tem como objetivo agrupar as soluções de alguns dos desafios PostgreSQL da plataforma [beecrowd](https://beecrowd.com/pt/)

## ORGANIZAÇÃO
Cada diretório denominado pelo número do desafio e seu título contém:
* Print do desafio
* Print do código solução + confirmação beecrowd
